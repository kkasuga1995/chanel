動作環境

OS

Windows Vista SP2以降に対応しています。


動作確認済

Windows Vista Ultimate SP2 32bit
Windows 7 Professional SP1 64bit
Windows 8.1 Pro Update 64bit
Windows 10 Pro 64bit


注意
.NET Framework 4.0(4.5/4.6)が必要です。
DupFileEliminatorはWPFアプリケーションです。
マウス拡張ツールを使用している場合、正常な動作が出来ない場合があります。
メモ
32bit/64bit両対応です。64bit環境で使用する場合は、64bitで動作します。
大量のファイルを一度に処理したり、大きな画像ファイルを扱う場合は、64bit環境での使用を推奨します。
